Definicao e uso tag <h1> a <h6>

As tags <h1> a <h6> são usadas para definir titulos HTML, <h1> define o titulo mais importante. <h6> define o menos importante

Nota: Use apenas um <h1> por página - deve representar o titulo/assunto principal de toda a pagina. Além disso, não pule os niveis de titulo - comece com <h1>, depois use <h2> e assim por diante

HEADER

O elemento <header> representa um grupo de introdução ou um conjunto de links de navegação. É comum que o elemento <header> contenha o título do documento, logotipo, informações de autoria, data de publicação e outros elementos introdutórios.

NAV

O elemento <nav> representa uma seção de navegação do documento. É usado para agrupar links de navegação, como menus, barras laterais ou rodapés.

MAIN

O elemento <main> representa o conteúdo principal do documento. Ele deve conter o conteúdo mais relevante e exclusivo da página, excluindo cabeçalhos, rodapés e barras laterais.

FOOTER

O elemento <footer> representa o rodapé de um documento ou seção. Ele geralmente contém informações sobre o autor, direitos autorais, links de contato e outras informações relacionadas ao documento.

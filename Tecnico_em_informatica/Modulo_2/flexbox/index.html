Trabalhando com FlexBox

O Flexbox (Flexible Box Module), é um modelo de layout unidimensional foi criado para tornar tarefas mais simples e funcionais: os filhos de um elemento com Flexbox podem se posicionar em qualquer direção e pode ter dimensões flexíveis para se adaptar.
